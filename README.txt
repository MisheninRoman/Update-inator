Инструкция по использованию программы "Обнавлятинатор"

 Что делает программа:
Автоматически нажимает кнопку "Обновить" в системе HPSM через браузер с заданным интервалом.

 Простые шаги:

 1. Подготовка
- Убедитесь, что в папке с программой есть файлы:
  - `yandexdriver.exe` (для Яндекс.Браузера)
  - `chromedriver.exe` (для Chrome)

 2. Запуск программы
- Запустите файл `program.exe` (дважды щелкните по нему)

 3. Настройка браузера
- Нажмите кнопку **"Яндекс.Браузер"** или **"Chrome"**
- Если браузер не нашел автоматически:
  - Нажмите **"Поиск браузера"**
  - Или укажите путь вручную через **"Обзор"**

 4. Запуск браузера
- Нажмите **"Запустить браузер"**
- Должен открыться браузер со страницей системы HPSM

 5. Поиск кнопки
- Нажмите **"Найти кнопку"**
- Если кнопка найдена - она подсветится красной рамкой
- Если не находит - попробуйте другой селектор в поле ввода

 6. Авто-клик
- Укажите интервал в секундах (например, 5)
- Нажмите **"Запустить авто-клик"**
- Программа будет автоматически нажимать кнопку "Обновить"

 7. Остановка
- Чтобы остановить: **"Остановить"**
- Чтобы закрыть браузер: **"Закрыть браузер"**

 Важно:
- Не закрывайте браузер вручную во время работы
- Не меняйте вкладки в браузере
- Закрывайте программу через крестик в углу окна

 Если что-то не работает:
1. Проверьте, что драйверы в папке с программой
2. Убедитесь, что браузер установлен
3. Проверьте подключение к интернету
4. Закройте и перезапустите программу

Всё готово! Программа будет автоматически обновлять страницу.
